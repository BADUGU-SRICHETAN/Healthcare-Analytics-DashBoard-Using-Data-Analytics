# Healthcare-Analytics-DashBoard-Using-Data-Analytics

 
# Project Description

Interactive Power BI dashboard for analyzing healthcare data, including patient admissions, disease trends, treatment outcomes, hospital performance, and patient demographics. The project provides insights into healthcare operations, patient care quality, and resource utilization through interactive visualizations and filters.

# Business Problem

The objective of this project is to analyze healthcare data to identify patient trends, monitor hospital performance, optimize resource allocation, and support data-driven decision-making for improving healthcare services and patient outcomes.

# Features
Patient Admissions Analysis
Disease and Diagnosis Analysis
Treatment Outcome Analysis
Patient Demographics Analysis
Hospital Department Performance
Length of Stay Analysis
Readmission Rate Analysis
Interactive Filters and Slicers
KPI Cards and Gauge Charts

# Tools Used
Power BI
Power Query
DAX
Excel / CSV Dataset

# Key Insights Generated
Most common diseases and diagnoses
Monthly and yearly patient admission trends
Department-wise patient volume and performance
Average length of hospital stay
Patient distribution by age and gender
Treatment success and readmission rates
Resource utilization across departments
Healthcare service performance metrics

# Project Objective
To develop an interactive healthcare analytics dashboard that enables hospitals and healthcare organizations to monitor patient data, evaluate treatment effectiveness, optimize operational efficiency, and improve overall healthcare delivery through data-driven insights.
